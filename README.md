# نظام إدارة بيانات الطلاب 

هذا المشروع هو عبارة عن تطبيق C# Console يعتمد النظام على بنية المعطيات Doubly Linked List (اللائحة المتصلة المزدوجة) لإدارة سجلات الطلاب.

## الميزات والمتطلبات التي تم تنفيذها:
- بنية المعطيات: تنفيذ كامل للائحة المزدوجة (Doubly Linked List) لتخزين بيانات الطلاب.
- بيانات الطالب: يحتوي كل سجل على:
    - الرقم الجامعي (فريد).
    - الاسم الكامل.
    - المحافظة.
    - علامتي الاختبار.
    - التقدير (باستخدام Enums: ممتاز، جيد جداً، جيد، راسب).
- العمليات الأساسية:
    - إضافة طلاب جدد في بداية أو نهاية اللائحة.
    - حذف سجل طالب عن طريق الرقم الجامعي.
- الخوارزميات المنفذة:
    - الفرز (Sorting): ترتيب الطلاب أبجدياً حسب الاسم أو تصاعدياً حسب المحصلة النهائية.
    - البحث العودي (Recursive Search): استخدام دالة عودية للبحث عن علامات محددة داخل اللائحة.
    - الفلترة: عرض قائمة الطلاب الذين تزيد محصلتهم عن 85.
- الحسابات: حساب المحصلة النهائية تلقائياً (متوسط علامتي الاختبار).

## كيفية التشغيل
1. افتح المشروع باستخدام برنامج Visual Studio.
2. قم بتشغيل ملف Program.cs.
3. واجهة البرنامج باللغة الإنجليزية لسهولة الاستخدام البرمجي.
